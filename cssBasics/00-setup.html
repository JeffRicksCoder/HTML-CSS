CSS Statements

CSS statements are made up of four main components:

  1. Declaration: the statement as a whole with curly braces.
  2. Selector:the name of the part you wan to style.
  3. Property: the properties to edit (e.g. width, height, background, ets.)
  4. Value: values that we assign (e.g. numbers, colors, urls, etc.)
